# Contributing

## Development setup

```bash
git clone https://github.com/salvacarrion/autonmt.git
cd autonmt/
pip install -e '.[dev]'
```

## Tests

```bash
pytest tests/
```

Unit tests cover enums, config merge, dataset paths, vocab, search/transformer parity, and
HF mapping. The functional suite includes a hermetic synthetic-corpus end-to-end test of
the preprocessing pipeline (raw → splits → SentencePiece → encoded) that runs in **under
two seconds** with no GPU — use it to verify refactors quickly:

```bash
pytest tests/functional/test_builder_e2e.py -v
```

## Lint

CI only fails the build on syntax errors and undefined names:

```bash
flake8 . --count --select=E9,F63,F7,F82 --show-source --statistics
```

## Building the docs locally

```bash
pip install -e '.[docs]'
mkdocs serve            # live-reload at http://127.0.0.1:8000
mkdocs build --strict   # catch broken internal links before pushing
```

The site is published automatically to GitHub Pages on every push to `main` that touches
`docs/`, `mkdocs.yml`, or `autonmt/` (the API reference is generated from docstrings). The
workflow is [`.github/workflows/docs.yml`](https://github.com/salvacarrion/autonmt/blob/main/.github/workflows/docs.yml).

### Writing docstrings

The API reference is autogenerated with [mkdocstrings](https://mkdocstrings.github.io/) using
**NumPy-style** docstrings. A new public class or function picks up documentation for free —
just write a clear docstring with `Parameters` / `Returns` sections.

## Conventions

A few project conventions worth knowing (see also `CLAUDE.md` in the repo root):

- **`assert` statements stay.** They catch bugs during research iteration; don't strip them.
- **Examples are self-contained.** Each `examples/*.py` duplicates its boilerplate on
  purpose — don't extract shared helpers.
- **Keep the core minimal.** Prefer extension points (callable hooks, subclassing) over new
  built-in flags.
- Dataset files use the **language code as the extension** (`train.es`), never as a
  directory.
- The config dump (`logs/config_{train,predict}.json`) keeps **primitives only** — arbitrary
  objects won't round-trip.
